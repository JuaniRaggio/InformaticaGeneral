# Manejo de strings

- Uso de *while + i* para recorrer strings y verificar caracter por caracter cierta/s condicion/es

- Pasaje $"strings" \rightarrow "numero (int() o float())"$. *TENER EN CUENTA QUE TODO LO QUE INGRESAN POR CONSOLA ES STRING*


# Validacion de datos

- Pedir datos hasta que se cumpla cierta condicion

- Leer numeros por consola (_en realidad son strings_) y operar con ellos (_pasaje de string a numero facilita las operaciones_)

- Validar si un numero ingresado esta dentro de un rango (considerar item anterior)

# Entendimiento de enunciado

- Si me dicen "se pasa un numero", "dado un numero"; *no es necesario verificar que sea un numero*

- *Ver ejemplos es importantisimo* para entender que casos hay que tener en cuenta.

- Si nos muestran un ejemplo en el que pedimos un dato (ya sea numero, frase, etc), nos pasan dato *incorrecto* y se vuelve a pedir el numero. Quiere decir que *tenemos que validar que el dato sea valido y volver a pedirlo hasta que lo sea (en caso de que pasen algo incorrecto)*

- Si me dicen *"el programa finaliza cuando se ingresa un dato invalido"*, entonces no tenemos que pedir hasta que sea valido, simplemente terminamos cuando es invalido y escribimos el mensaje de error si es que lo piden
