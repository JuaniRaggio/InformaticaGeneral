# Ejercicio 1

Se desean procesar numeros ingresados por consola

## Ingreso de numeros

- El usuario ingresa numeros separados por espacios
- Se van a pedir numeros mientras sea valido el formato solicitado
- Si el formato en el que se envian los numeros es incorrecto, termina el programa y se muestra un mensaje de error

## Validacion
El formato es valido si:

- Los numeros estan separados entre si por *espacios*
- Contiene numeros enteros o flotantes (se separa el decimal con ".")
- Se debe ingresar al menos un numero
- Se pueden ingresar como mucho 5 numeros distintos (pueden ser numeros de mas de un digito)
