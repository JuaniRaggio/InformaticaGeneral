## frase_interminable

### Dependencias
Requiere python-tk instalado.

Instalación:
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# macOS
brew install python-tk

# Windows
pip install tk
```

### Ejecución
```bash
python frase_interminable.py
```

### Uso
- Escriba texto en el campo de entrada
- Presione "Enviar" o Enter para agregar la frase
- Use "Pedir siguiente frase" para obtener sugerencias
- Las frases se acumulan en el área de texto principal
